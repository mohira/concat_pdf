# 複数のPDFを1つにするツール

## Usage

```sh
$ concat pdfs 
Combined .../concat/output/merged_2022-01-07 21:09:29.108089.pdf
```

```sh
$ concat pdfs --dst tmp
Combined tmp/merged_2022-01-07 21:09:53.262776.pdf
```

```sh
$ concat --help
Usage: concat [OPTIONS] SRC

  対象ディレクトリ以下のPDFを1つにまとめる

Options:
  --dst PATH  出力ディレクトリ
  --help      Show this message and exit.
```
